# ShopHub - E-Commerce Product Listing

A fully functional e-commerce product listing web application built with Flask and SQLite.

## Features

- Product listing with pagination
- Search and filter by category
- Product detail pages
- Add/delete products (admin functionality)
- Responsive design
- REST API endpoints
- Rating and review display
- Stock management
